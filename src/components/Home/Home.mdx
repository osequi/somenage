# Somenage

An interactive design system for telling stories.

## Features

1. Renders Semantic HTML5 code validated by W3C.
2. Aligns elements to the typographic grid.

## Built on

- Next.js, for static rendering.
- React, for interactivity.
- Typescript, for scalable code.
- Emotion, for style objects and functions.
- Jest, for testing theme, hooks and components.
- JsDoc, to generate API documentation.
